Keyboard ShortCuts for Mac Os
=============================

Mac Icons
---------
 ⌃⌥⇧⌘
 ⎋⏏⌫↑⇡↖⇞⇥

`⇧` => `SHIFT`

`⌃` => `CONTROL`

`⌥` => `ALT` or `OPTION`

`⌘` => `COMMAND`

Go to spotlight 
---------------
 `⌃SPACE` 

Copy and Move
-------------
`⌘C`,  `⌥⌘V`,
